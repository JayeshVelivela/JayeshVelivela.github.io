# Quick Setup Guide

This portfolio already includes the required public assets:

- `public/jayesh.jpg`
- `public/Jayesh_Velivela_Resume.pdf`

Run the site locally:

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

Build the static GitHub Pages export:

```bash
npm run build
```

The exported site is generated in `out/`.
