# SEO and Sharing Guide

The portfolio is configured for:

```text
https://jayeshvelivela.github.io
```

## What Is Already Set

- Page title and description in `app/layout.tsx`
- Open Graph metadata for link previews
- Twitter card metadata
- Search indexing enabled
- `public/robots.txt`
- Resume and public links updated to the June 2026 portfolio content

## After Deployment

1. Visit `https://jayeshvelivela.github.io` and confirm the site loads.
2. Check that `public/jayesh.jpg` appears in the hero.
3. Check that `public/Jayesh_Velivela_Resume.pdf` downloads correctly.
4. Share the URL on LinkedIn, your GitHub profile, and your resume.

## Optional Polish

- Add a custom domain later if you buy one.
- Add a generated social preview image if you want richer link cards.
- Keep `lib/portfolio.ts` current whenever your resume changes.
